Lung cancer detection system using transfer learning

I have developed an automatic system that helps in the detection of abnormalities in the lung and helps to early diagnose of cancer using machine learning. the LC25000 dataset is used for detection.

  • Python, tensorflow, Opencv , Numpy are used.
  • Preprocessing techiques: Histogram equalization, watershed segmentation, Data augmentation
  • Feature extraction: VGG19 and ResNet50 (transfer learning)
  • classification : feed forward netural network
  • Results: 97.6% accuracy when VGG19 is used as Feature extractor and 99% accuracy when ResNet50 is used as feature extractor

fingerprint recognition system

In this project, a novel method of using four preprocessing techniques—histogram equalization, Gabor filter, thinning, and cropping—is used to improve the accuracy of the machine learning algorithms. the Contact-based 2D Fingerprint Images Database is used for detection.

  • Python, tensorflow, Opencv , Numpy are used.
  • Preprocessing techiques: Histogram equalization, watershed segmentation, Data augmentation
  • Feature extraction: VGG19 and ResNet50 (transfer learning)
  • classification : feed forward netural network
  • Results: 97.6% accuracy when VGG19 is used as Feature extractor and 99% accuracy when ResNet50 is used as feature extractor
